Volume Select

This page allows you to explore recreated versions of the SBS ("I'm Taking Questions") sections from Eiichiro Oda's One Piece manga. These segments contain direct fan questions and Oda's responses. Our goal is to provide access to these interactions along with translated summaries and character tallies.
